- # Learning Logseq
	- Finally found a written guide to Logseq here: https://docs.logseq.com/#/page/contents
	- To add a commonly-used alias, use a page alias. In the **first block** of the page, add the following line: `alias:: name`, where `name` is the alias you want to use for the page.
	- The documentation says that you can override a page's display name to be different from the file name with `title:: name`, but all this does is rename the file to match the new display name, which seems useless because I could just rename the file if that's what I wanted. https://github.com/logseq/logseq/issues/4722#issuecomment-1482850848
	- Logseq has three optional, built-in priorities that can be added to blocks with the command `/A`, `/B`, and `/C`
	- You can add deadlines to blocks with the command `/deadline`
	- You can add scheduled dates to blocks with the command `/scheduled`
